# Experiment Rigor Checklist

- [x] Runnable baseline script.
- [x] Fixed random seed for reproducible baseline metrics.
- [x] Stronger failure-case stress against the hidden reliable-mode assumption.
- [x] Baseline and proposed method reported with both collision and blocked-step metrics.
- [x] Negative result integrated into manuscript limitations.
- [ ] Multiple independent seed confidence intervals.
- [ ] Hardware or high-fidelity simulation.
- [ ] Broader controller families.
- [ ] Formal proof beyond the toy one-step inequality.

Decision: adequate for a narrow workshop mechanism paper, not for a full empirical robotics submission.
